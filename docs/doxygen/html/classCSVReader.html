Detailed Description

CSV reader implementation using fast-cpp-csv-parser.

Format specifics:

  • Expects RFC-style CSV with commas and double-quote escaping.
  • Expected header columns (in the CSV): id,title,status,priority,created_date, description,assignee,due_date,tags
  • tags is returned as a single field; callers should expect a comma- separated tag string in that field (the reader does not split tags for you).
