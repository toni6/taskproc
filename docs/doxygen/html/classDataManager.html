Detailed Description

DataManager manages the loading and reloading of tasks from files.

Use this class to load tasks from supported file formats (CSV, JSON, etc.).

◆ load_from_file()

bool DataManager::load_from_file ( std::string_view filepath)

Load tasks from filepath and replace the manager's tasks on success.

Precondition
filepath is a path to a readable file and a matching reader exists.
Postcondition
On success: tasks_ contains the loaded tasks and current_filepath_ equals filepath.
On failure: tasks_ remains unchanged.
Exceptions
std::exception(or implementation-specific exceptions) on I/O or parse errors.
Parameters
filepathPath to the file to load.
Returns
true if the file was successfully loaded and parsed, false otherwise.
