Detailed Description

Persisted view state: filepath + history of view-modifying commands.

ViewStorage stores in-memory state and optionally persists it to a small storage file ("./.taskproc.storage"). The state contains the path to the last-loaded tasks file and an ordered list of view-modifying actions that should be replayed on top of the file to reconstruct the current view.
